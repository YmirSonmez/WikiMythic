## Description
Will drop an an from the plugin `PhatLoot`.


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| type      | t         | The object to drop                                                   |         |


## Examples
```yaml
  Drops:
  - phatloot LootTableName 1
```
```yaml
  Drops:
  - phatloot{t=LootTableName} 1
```
